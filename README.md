# Crypto-parser
Parser for prices of cryptocurrencies. 
Parser takes prices and tickers from https://coinmarketcap.com/
## Table with cryptocurrencies
The table with cryptocurrencies load from Google gspread: https://docs.google.com/spreadsheets/d/1ZBdnihibZ3_g0coJhQwPzNcsRECEsuvxw0o768HwzV4/edit?usp=sharing

This table contains columns:
- "coin", which contains names of cryptocurrency
- "ticker", which contains ticker of cryptocurrency
- "purchase_value", which contains amounth of invested money (usd)
- "amount", which contains amounth of cryptocurrency
